# Watchtower
  Watchtower presents a smart security system based on a Raspberry Pi 5 with a mounted webcam and servo controls. The System has AI detection of humans or pets and sends the data to a cloud web dashboard and database, ready to be consulted from anywhere by the user. The goal of this project is to showcase a relatively low-cost and scalable means of keeping track of access in a private area like a home or small business.
